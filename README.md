# openlayers

Rendering GPX tracks with OpenLayers for Garmin eTrex SE.

## Run

Start server:

```
npm run start
```

Create screenshots:

```
npm test
```


## Examples

Specific location:

http://localhost:3000/?lat=1.3095707398376149&lon=103.8943942558364

Rotate the view:

http://localhost:3000/?lat=1.3095707398376149&lon=103.8943942558364&rotation=180
